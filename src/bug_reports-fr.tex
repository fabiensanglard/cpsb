\chapter*{Signaler des erreurs}

Ce livre vise l'exactitude. Si vous repérez des erreurs, des oublis ou des fautes de frappe, merci de prendre le temps de les signaler.

Encore mieux : proposez une correction via une pull request, puisque le code source complet est disponible en ligne sur GitHub.

Signaler un problème : \textbf{\href{https://github.com/fabiensanglard/cpsb/issues}{https://github.com/fabiensanglard/cpsb/issues}}.

Envoyer une pull request : \textbf{\href{https://github.com/fabiensanglard/cpsb/pulls}{https://github.com/fabiensanglard/cpsb/pulls}}.

M'envoyer un e-mail : \textbf{\href{mailto:fabiensanglard.net@gmail.com}{fabiensanglard.net@gmail.com}}.
